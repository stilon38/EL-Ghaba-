# الغابة · التعرّف على الأشخاص (EL-Ghaba Recognition)

تطبيق ويب للتعرّف على الأشخاص من **الوجه** و**الصوت** — يعمل بالكامل داخل متصفحك،
بدون أي خادم، ويتعامل مع **مئات الأشخاص** عبر قاعدة بيانات محلية.

> A privacy-first, in-browser web app that recognizes people by **face** and **voice**.
> Everything runs locally in the browser — no server, no cloud, no data leaves the device.

---

## ✨ المميزات

- 🔍 **تعرّف مباشر على الوجوه** من الكاميرا مع رسم مربّعات وأسماء وثقة التطابق في الوقت الحقيقي.
- 🎙️ **تعرّف على المتحدث بالصوت** عبر بصمة صوتية مبنية على معاملات MFCC.
- ➕ **تسجيل الأشخاص**: التقاط عدة عيّنات وجه + تسجيل بصمة صوت لكل شخص.
- 👥 **إدارة قاعدة البيانات**: عرض، حذف، وتصدير/استيراد نسخة احتياطية (JSON).
- 🔒 **خصوصية كاملة**: كل البيانات تُخزَّن محلياً في متصفحك (IndexedDB). لا يُرسَل أي شيء للإنترنت.
- 📱 يعمل على الهاتف والكمبيوتر، وواجهة عربية بالكامل (RTL).

## 🧠 كيف يعمل؟

| الوسيلة | التقنية | التفاصيل |
|--------|---------|----------|
| الوجه | [`@vladmandic/face-api`](https://github.com/vladmandic/face-api) (TensorFlow.js) | كشف الوجه + استخراج متجه مميّز 128 بُعد، ومطابقة بأقرب جار (مسافة إقليدية). |
| الصوت | [Meyda](https://meyda.js.org/) MFCC | متوسط + انحراف معياري لمعاملات MFCC (26 بُعد) كبصمة صوت، ومطابقة بتشابه جيب التمام. |
| التخزين | IndexedDB (`idb`) | يتّسع لمئات الأشخاص محلياً مع دعم النسخ الاحتياطي. |

النماذج مضمّنة داخل `public/models` (~13MB) فيعمل التطبيق **دون اتصال بالإنترنت** بعد أول تحميل.

## 🚀 التشغيل

```bash
npm install          # تثبيت الاعتماديات
npm run dev          # تشغيل بيئة التطوير على http://localhost:5173
```

للبناء للإنتاج:

```bash
npm run build        # يُنتج مجلد dist/ جاهز للنشر
npm run preview      # معاينة النسخة المبنية
```

> ⚠️ الكاميرا والميكروفون يتطلبان سياقاً آمناً: `localhost` أو موقعاً عبر **HTTPS**.

إذا احتجت لإعادة نسخ النماذج يدوياً بعد `npm install`:

```bash
npm run fetch-models
```

## 📖 طريقة الاستخدام

1. **تسجيل**: افتح تبويب «تسجيل»، اسمح بالكاميرا/الميكروفون، التقط 3–5 عيّنات وجه بزوايا مختلفة، وسجّل بصمة الصوت، ثم أدخل الاسم واحفظ.
2. **تعرّف**: افتح تبويب «تعرّف» لرؤية التعرّف المباشر على الوجوه، أو اضغط «استمع وتعرّف» للتعرّف بالصوت.
3. **الأشخاص**: أدر قاعدة البيانات وصدّر/استورد نسخة احتياطية.

## 🗂️ البنية

```
src/
  lib/
    db.ts        قاعدة بيانات IndexedDB (الأشخاص + النسخ الاحتياطي)
    face.ts      تحميل نماذج الوجه والكشف واستخراج المتجهات
    voice.ts     التسجيل من الميكروفون وبصمة الصوت (MFCC)
    match.ts     دوال المسافة/التشابه والمطابقة بأقرب جار
  hooks/
    useMediaStream.ts   إدارة تدفق الكاميرا/الميكروفون
  components/
    RecognizePage.tsx   التعرّف المباشر (وجه + صوت)
    EnrollPage.tsx      تسجيل شخص جديد
    PeoplePage.tsx      إدارة الأشخاص
  App.tsx        التنقّل بين التبويبات
public/models/   نماذج face-api المضمّنة
```

## ⚙️ ملاحظات حول الدقة

- **الوجه** دقيق جداً (نموذج مُدرَّب). زد عدد عيّنات كل شخص لتحسين الثبات مع تغيّر الإضاءة والزاوية.
- **الصوت** يعتمد طريقة MFCC كلاسيكية؛ فعّال للتمييز بين الأصوات في بيئة هادئة، لكنه أقل قوة من نماذج بصمة الصوت العميقة. سجّل الصوت في مكان هادئ لأفضل النتائج، ويمكن ضبط العتبات في `src/lib/match.ts`.

## 🔐 الخصوصية والاستخدام المسؤول

هذا التطبيق يُخزّن البيانات البيومترية (الوجه/الصوت) محلياً على جهازك فقط. عند استخدامه للتعرّف على أشخاص آخرين، احصل على موافقتهم والتزم بالقوانين المعمول بها لحماية البيانات.
